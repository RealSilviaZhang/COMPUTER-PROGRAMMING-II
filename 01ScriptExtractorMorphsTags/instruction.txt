Assignment Instructions
We want to create a script to extract all morpheme and parts of speech from a given morpheme analysis corpus.
A function (get_morphs_tags) that extracts morphemes and parts of speech is created and given source code (get_morphs_tags).py) complete.

The method of executing the completed script is as follows.
$ ./get_morphs_tags.py Input File > Output File

Ex) $ ./get_morphs_tags.py 2019.tag > 2019.txt

* Morpheme Analysis Corpus in Attachment (tagged.zip)
