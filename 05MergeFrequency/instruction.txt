Assignment Instructions
We want to create a script that merges annual frequency files.
A function (merge_k_sorted_freq) that merges word frequencies is created to complete a given source code (merge_k_frequency.py).

The method of executing the completed script is as follows.
$ ./merge_k_frequency.py input file(s)

Ex) 
  $ ./merge_k_frequency.py *.freq > result

* Heapq module must be used (dictionary cannot be generated)
